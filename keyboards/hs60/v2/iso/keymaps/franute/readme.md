Custom keymap for ISO HS60 V2 ISO. VIA support disabled.
=========================================================

![Layout image](https://imgur.com/a/9mMN675)

Base layer is basically the default but adding arrow keys when tapping lower-right keys of the keyboard.
Menu key is also moved to accomodate a second layer.
All keys in layers are set to "transparent" unless specified.

First layer (right of alt gr):
    - Fn keys in numbers.
    - Volume management:
        - up -> tab
        - down -> caps lock
        - mute -> q
    - additional arrow keys: a s d w
    - Backspace -> delete
    - Enter -> insert
    - left bracket -> home
    - ' -> end
    - right bracket -> page up
    - # -> page down
    - p -> print screen
    - menu key (left of right control):
        menu when tap
        hold for Layer 3

Second layer (with the menu key - left of right control, inherits from base layer):
    - default RGB functions from the default ISO keymap
    - music track control:
        - next track -> tab
        - prev track -> caps lock
        - play / pause -> q

Third layer (inherits keys from layer 1):
    - right bracket -> reset
